# Virual-Mouse

# How to setup and run

  ### Pre-requisites
  
  Python: (3.6 - 3.8.5)<br>
  Anaconda Distribution: To download click [here](https://www.anaconda.com/products/individual).
  
  
  Step 1: 
  ```bash
  conda create --name gest python=3.8.5
  ```
  
  Step 2:
  ```bash
  conda activate gest
  ```
  
  Step 3:  
  ```bash
  pip install -r requirements.txt
  ```
  
  Step 4:
  ```bash 
  python Virtual_Mouse.py
  ```
